Información del Ticket Cerrar

¿Cómo funciona el sistema de Tickets de Soporte?

Para ayudar con cualquier problema o consulta, puedes crear un Ticket de Soporte. A continuación te explicamos cómo funciona:

  1. Enviar un Ticket: Al hacer clic en "Enviar Ticket de Soporte", podrás describir el problema que estás experimentando.
  2. Seguimiento: Después de enviar tu ticket, nuestro equipo revisará tu solicitud y te proporcionará una respuesta.
  3. Gestión de Tickets: Los usuarios avanzados pueden gestionar sus tickets y ver el estado de los mismos.
  4. Base de Conocimientos: Accede a la base de conocimientos para encontrar soluciones rápidas a problemas comunes.

Si eres administrador, tendrás opciones adicionales para gestionar usuarios y ver todos los tickets.
